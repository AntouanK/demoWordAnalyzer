Demo test!

Write an application that processes a paragraph of text and outputs all the words found in the text and the number of occurrences of each word. The words need to show up in the results list just once and they must be sorted alphabetically and written all in lower caps.

Example:
Input:
It was so and so.
Output:
and 1
it 1
so 2
was 1

The application can be written in JavaScript or CoffeeScript and use whatever 3rd party libraries required. User interface design is up to the developer and is not important as long as it functional. Test expected to take anywhere between a couple of hours and a day, depending on how much the candidate wants to show.

Guidelines:
The candidate is expected to demonstrate:
Test driven development approach
Good use of an MVC framework (Knockout, Backbone, Ember etc.)
Design patterns knowledge
Code quality (separation of concerns, decoupling etc.)

Bonus:
Render the results graphically
Animate the graphical representation
